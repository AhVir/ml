K-Means Algorithm

function KMeans(data, k)
// Initialize k centroids
centroids ← initialize(data, k)
repeat
// Assignment Step
for each point in data
distances ← compute_distances(point, centroids)
assign point to nearest centroid
end for
// Update Step
old_centroids ← centroids
for each cluster in k
centroids[cluster] ← mean(assigned_points)
end for
// Check convergence
if centroids == old_centroids
break // Converged
end if
until convergence
return centroids, assignments
end function

Complexity

Time: O(n·k·i·d)
n=points, k=clusters
i=iterations, d=dimensions
Space: O(n + k)
