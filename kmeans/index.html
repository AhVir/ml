About K-Means

K-means is an unsupervised clustering algorithm that partitions n observations into k clusters by minimizing within-cluster variance.

How it works:
  1. Initialize K random centroids
  2. Assign each point to nearest centroid
  3. Recalculate centroids as cluster means
  4. Repeat until convergence
